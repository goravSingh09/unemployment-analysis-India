# Unemployment Analysis with Python

## Project Overview
This project analyzes unemployment trends in India using Python, with a special focus on the impact of COVID-19 on unemployment rates.

The analysis helps identify:
- Changes in unemployment over time
- Most affected states
- COVID-19 impact on employment

## Dataset
The dataset contains unemployment statistics across different regions of India, including:
- Region
- Date
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)
- Longitude
- Latitude

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Visual Studio Code

## Project Workflow
1. Data Collection  
2. Data Cleaning  
3. Missing Value Analysis  
4. Data Visualization  
5. Trend Analysis  
6. Conclusion  

## Visualizations
This project includes:
- Unemployment Rate by Region (Bar Chart)
- Unemployment Rate Over Time (Line Graph)
- Top 5 States with Highest Unemployment

## Key Insights
- Unemployment increased sharply during COVID-19.
- Peak unemployment occurred around May–June 2020.
- Highest affected states:
  - Haryana
  - Tripura
  - Jharkhand
  - Bihar
  - Delhi

## Conclusion
COVID-19 significantly impacted employment in India.  
Using Python data analysis libraries, we successfully analyzed unemployment trends and identified the most affected regions.
